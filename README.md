# React Native Test Project

React & React Native boilerplate project with TypeScript

## Stack

* TypeScript
* React
* Recompose
* Redux
* Redux-promise
* Prettier
* Jest

### Web

* Webpack v4
* ts-loader
* Babel

### Native

* ts-node
* React Native

## Require

* yarn
* Android SDK
* node

## Develop

### Native(Android)

```bash
yarn install
yarn build:watch
yarn android
```

### Web

```bash
yarn install
yarn web-start
```
